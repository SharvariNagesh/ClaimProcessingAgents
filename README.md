# Insurance Claim Processing App

This is a Streamlit web application for processing automobile insurance claims.

## Features

- Load claim PDF from S3 bucket
- Extract and summarize claim details
- Automatically approve claims
- Display results in the UI

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`

## Requirements

- AWS credentials configured for S3 access
- Python 3.7#
